왜 Rust인가요?

메모리 안전성

컴파일 시점에 널 포인터, 데이터 경쟁을 방지하여 안전한 코드를 작성합니다.

뛰어난 성능

C/C++와 동등한 수준의 성능으로 시스템 프로그래밍에 최적화되어 있습니다.

생산적인 개발

강력한 타입 시스템과 Cargo를 통해 효율적인 개발 경험을 제공합니다.

어떻게 기여할 수 있나요?

Rust 한국어 문서 프로젝트는 여러분의 참여로 성장합니다. 번역, 새로운 문서 작성, 기존 문서 개선 등 다양한 방법으로 기여할 수 있습니다.

  1. GitHub 저장소를 Fork하고 Clone합니다.
  2. 새로운 브랜치를 생성하고 문서를 번역하거나 작성합니다.
  3. 변경 사항을 커밋하고 Pull Request를 보냅니다.

더 자세한 내용은 기여 가이드라인을 참조해주세요.
