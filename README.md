# Rust 한국어 문서 프로젝트

이 프로젝트는 Rust 프로그래밍 언어의 공식 문서를 한국어로 번역하고, 한국어 사용자들을 위한 추가적인 자료를 제공하기 위해 시작되었습니다. 모든 Rust 사용자들의 참여를 환영합니다!

## 참여 방법

문서 기여에 대한 자세한 내용은 [CONTRIBUTING.md](CONTRIBUTING.md) 파일을 참조해주세요.

## 문서 구조

- `index.html`: 메인 페이지
- `docs/`: Rust 한국어 문서 파일들이 위치할 디렉토리
  - `docs/introduction.md`: Rust 소개
  - `docs/getting-started.md`: 시작하기
  - ...

## 라이선스

이 프로젝트의 모든 문서는 MIT 라이선스 하에 제공됩니다. 자세한 내용은 `LICENSE` 파일을 참조해주세요.
